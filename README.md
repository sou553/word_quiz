# 単語トレーニングサイト

GitHub Pagesで公開できる静的な四択単語テストです。

## 収録単語群

- システム英単語: 1,200語
- 英検準1級: 1,900語
- 全単語: 3,100語

## 主な機能

- 単語群の選択
- 問題数: 10 / 30 / 50
- 英語→日本語 / 日本語→英語 / 両方ランダム
- 四択問題
- 最初は選択肢を非表示
- 出題範囲指定
- 選択肢は出題範囲ではなく、選択中の単語群全体から作成
- 前回までのミスだけ再テスト
- ダークモード
- キーボード操作

## GitHub Pagesでの公開

1. リポジトリに `docs` フォルダをアップロードします。
2. GitHubの `Settings` → `Pages` を開きます。
3. `Branch: main`、`Folder: /docs` を選択します。

## さらに単語群を追加する方法

CSVは `単語,意味` の2列にしてください。
`regenerate_words.py` の `SOURCES` にCSVファイル名・単語群ID・表示名を追加して実行すると、`docs/words.js` が再生成されます。

## 単語群

- システム英単語：1,200語
- 英検準1級：1,900語
- 1900語リスト：1,900語
- 全単語：5,000語

`docs/1900_words_meanings.csv` を追加し、`docs/words.js` に反映済みです。
CSVを更新した場合は、`python regenerate_words.py` で `words.js` を再生成できます。
