使い方

  • 単語群を選び、範囲を指定して出題できます。
  • 選択肢は出題範囲ではなく、選択中の単語群全体から作成されます。
  • ミスした単語はブラウザに保存され、「前回までのミスだけ出題」で復習できます。
  • キーボード操作：Spaceで選択肢表示、1〜4で回答、Enterで次へ。
